# 司法鉴定系统使用手册验收矩阵

更新时间：2026-06-12

## 1. 验收依据

本项目当前验收依据为：**《司法鉴定系统使用手册.docx》及 `docs/judicial-appraisal-flowchart-baseline.md` 登记的主流程图、细化流程图。**

验收流程范围按 `docs/judicial-appraisal-manual-based-requirements-v2.md` 第 9 章 9.1 至 9.19 执行，共 **19 个验收流程**。后续逐流程跑通应主要对照 `docs/flowcharts/` 中的流程图；流程图中存在但使用手册未列明的流程（如案件暂停）先作为差异项记录，不直接扩展当前必要验收指标。
