# 原有 OA 样例数据

更新时间：2026-06-13

本目录保存从原有 OA 测试环境导出的工作查询样例，用于司法鉴定系统重建过程中的测试夹具、字段对照、报表对照和流程等价性校验。

## 数据来源

- 来源系统：原有 OA 测试环境
- 来源页面：流程中心 -> 工作查询
- 查询范围：全部范围
- 查询状态：所有状态
- 截止日期：2026-06-13
- 登录凭据、会话 Cookie 不保存到本仓库

## 目录说明

- `exports/`：原有 OA 自带“导出工作列表”生成的 `.xls` 原始文件。
- `json/`：工作查询 jqGrid 接口返回的原始 JSON，包含表头定义和行数据。
- `csv-preview/`：由 JSON 行数据清洗 HTML 后生成的 CSV 预览，便于快速查看和人工筛样例。
- `metadata/old-oa-work-query-export-manifest.json`：`.xls` 导出文件清单、导出参数、响应头和文件大小。
- `metadata/old-oa-work-query-json-summary.json`：JSON/CSV 样例条数汇总。
- `raw/`：早期手工抓取的原始 OA 样例。

## 使用规则

- 这些数据仅用于本项目内部测试、验收和字段对照。
- 编写自动化测试前，应先从 `json/` 或 `csv-preview/` 中抽取代表性样例，再脱敏姓名、手机号、身份证号、地址、案号敏感片段和附件内容。
- 不要把原始样例直接作为公开文档、演示数据或外部交付物。
- 如果旧 OA 数据与使用手册或流程图不一致，先记录差异，再按使用手册和流程图优先处理。
- 若需要重新导出，应更新本文件的更新时间，并同步更新 `metadata/` 下的 manifest/summary。
